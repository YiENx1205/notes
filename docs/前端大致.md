# ECMAScript 6

ECMAScript 6.0(简称 ES6)是 JavaScript 语言的下一代标准（规范）

## 基本语法

### let 声明变量

```javascript
// var 声明的变量没有局部作用域
// let 声明的变量  有局部作用域
{
    var a = 1;
    let b = 2;
}
console.log(a)	// 0
console.log(b)  // b is not defined;在代码块外部里，就不是局部了
```

```javascript
// var 可以声明多次
// let 只能声明一次
var m = 1;
var m = 2;
let n = 10;
let n = 20;  
console.log(m) // 2
console.log(n) //'n' has already been declared
```
