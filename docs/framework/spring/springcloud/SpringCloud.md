# 简介

SpringCloud 是一套微服务技术的集合体

- 服务注册与发现
	- Eureka:x:
	- Zookeeper
	- Consul
	- **Nacos**

- 服务调用
	- **Ribbon**
	- LoadBalancer
	- 服务调用2
		- Feign:x:
		- **OpenFeign**

- 负载均衡
- 服务熔断和降级
	- Hystrix:x:
	- resilience4j
	- **sentienl**

- 服务消息队列

- 配置中心管理
	- Config:x:
	- apollo
	- **Nacos**

- 服务网关
	- Zuul:x:
	- **gateway**

- 服务总线
	- Bus:x:
	- **Nacos**
- 服务监控

- 全链路追踪

- 自动化构建部署

- 服务定时任务调度操作
